# Validator Lean Canvas — Cheat Sheet

## What It Is

A filled-in lean canvas for an "idea validation as a service" business model. Both a specific business model AND a reusable template.

## The Canvas

### Problem
Founders build on packaging assumptions. They ask friends if an idea is "good" and hear "yes" because politeness is the default. By the time they discover nobody will pay, they've burned runway.

### Solution
Automated market experiments that test willingness to pay before you build. Submit idea → generate packaging variants → deploy as subdomains → run paid acquisition → collect real economic signals → validate winner.

### Unique Value Proposition
**"We test if they'll pay before you build."**

### Customer Segments
1. **Early-stage founders** (pre-seed, seed) — need validation before spending $50K on dev
2. **Solo founders/bootstrappers** — can't afford 3 months building wrong thing
3. **Intrapreneurs** — testing internal ideas before resourcing
4. **Product teams pre-launch** — finding market fit before GA

### Revenue Streams
- **Pay-per-validation:** $149-499 per cycle
- **Subscription:** $99-299/mo for unlimited validations
- **Enterprise white-label:** incubators/accelerators/VCs
- **Consulting:** premium hands-on guidance

### Key Metrics
- Experiments run per founder per quarter (velocity)
- Time from idea to first validated signal (speed)
- Conversion rate per packaging (quality)
- Cost per validated idea (efficiency)
- Retention: founders running 2+ cycles (stickiness)

### Cost Structure
- **Variable:** Ad spend (40-60%), infrastructure (10-15%)
- **Fixed:** Core team, LLM inference, payment processing
- **Margin lever:** Better packaging generation = lower cost per validated experiment

### Unfair Advantage
Self-improving feedback loop. Every experiment creates data. Dataset trains better packaging generation. Competitors can build similar infra but can't replicate the proprietary dataset from thousands of experiments.

### Unit Economics
- Revenue per validation: $200-500
- Margin: 40-50%
- Scale: millions of founders globally, most validate multiple ideas
