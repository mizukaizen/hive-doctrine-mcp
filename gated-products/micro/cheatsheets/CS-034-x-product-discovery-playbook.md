# X Product Discovery Playbook — Cheat Sheet

## What It Is

Search-driven product discovery on X/Twitter. Find people publicly begging for products that don't exist, then build them.

## Core Insight

Recurring public complaints in searchable channels = validated demand = product opportunity.

## 6 Core Search Queries

### Tier 1: Unmet Need Signals (Highest Intent)

**Query 1: "I wish there was"**
```
(i wish there was) min_faves:50 lang:en
```
People actively articulating a missing solution.

**Query 2: "Why is there no"**
```
(why is there no) min_faves:50 lang:en
```
Frustration with a gap in the market.

**Query 3: "Someone should build"**
```
(someone should build) min_faves:30 lang:en
```
Explicit product demand signals.

### Tier 2: Pain Point Signals (High Intent)

**Query 4: "I hate [tool/process]"**
```
("i hate" AND [category keyword]) min_faves:20 lang:en
```
Emotional dissatisfaction with existing solutions.

**Query 5: "Switched from [tool] because"**
```
(switched from [tool] because) min_faves:10 lang:en
```
Reveals specific failure points of existing products.

**Query 6: "Looking for alternative to"**
```
(looking for alternative to) min_faves:10 lang:en
```
Active buyer intent — someone ready to switch.

## 3 Exploits

1. **Engagement as validation** — 50+ likes on a complaint = many people feel this way
2. **Reply mining** — the replies often contain more specific pain points than the original tweet
3. **Profile clustering** — if the same type of person (e.g. SaaS founders) keeps making the same complaint, that's your ICP

## 72-Hour Launch Framework

| Hour | Action |
|------|--------|
| 0-8 | Run all 6 queries. Capture 50+ high-signal tweets. Identify top 3 product opportunities. |
| 8-24 | Pick one opportunity. Build landing page. Write copy using the exact language from the tweets. |
| 24-48 | Reply to original tweeters: "I built this." Share landing page in relevant communities. |
| 48-72 | Measure signups/payments. >5% conversion = validated. <2% = try next opportunity. |

## Key Rules

- **Use their words** — landing page copy should mirror the exact language people used in tweets
- **50+ likes = signal** — below that it's noise
- **Speed beats perfection** — ship in 72 hours, not 72 days
- **Reply, don't cold pitch** — respond to the complaint, don't spam
