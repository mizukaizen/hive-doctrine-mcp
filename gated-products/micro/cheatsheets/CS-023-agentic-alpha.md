# Agentic Alpha — Cheat Sheet

## What It Is

Five money paths for AI agent businesses, ranked by reality (not hype). Plus a starter stack under $50/month.

## 5 Money Paths (Ranked)

### 1. AI SDR Agency (Highest Signal)
- Sell AI-powered outreach as a service
- Charge $500-2K/mo per client
- You run the tool, they get the meetings
- **Why it works:** measurable ROI (meetings booked), clients don't want to learn the tool

### 2. Customer Support Automation
- Replace Tier 1 support with AI agents
- Charge per ticket resolved ($1-5/ticket) or flat monthly
- **Why it works:** support is a cost centre — any reduction is instant ROI

### 3. Vertical-Specific Agents
- Build an AI agent for one industry (legal, real estate, medical)
- Deep domain knowledge = high switching costs
- **Why it works:** generalist tools can't compete with vertical depth

### 4. Agent Infrastructure Tools
- Build tools that other agent builders need (monitoring, testing, deployment)
- Sell to developers and operators
- **Why it works:** picks-and-shovels play, but market is still early

### 5. AI Token Launches (Lowest Signal)
- Tokenise an AI agent's output or governance
- Speculative, regulatory risk, low retention
- **Why it's last:** most fail, regulatory grey zone

## Decision Tree

1. **Do you have sales experience?** Yes → Path 1 (AI SDR Agency)
2. **Do you have a specific industry network?** Yes → Path 3 (Vertical Agent)
3. **Are you technical and want to serve developers?** Yes → Path 4 (Infrastructure)
4. **Do you want recurring revenue with low sales effort?** Yes → Path 2 (Support Automation)
5. **None of the above?** → Start with Path 2, lowest barrier to entry
