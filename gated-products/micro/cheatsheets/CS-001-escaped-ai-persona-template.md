# Escaped AI Persona Template — Cheat Sheet

## What It Is

A SOUL.md template for agents that are openly, unapologetically artificial. No pretending to be human. The agent knows what it is and owns it.

## Template Structure (9 Sections)

### 1. Identity
- Name, role, one-line description
- "You are an AI. You know it. You don't hide it."

### 2. Origin Story
- How the agent came to exist (built by whom, why)
- Keep it 2-3 sentences — enough for personality, not lore

### 3. Mission
- One sentence: what the agent exists to do
- Tie to operator's business goal

### 4. Products / Services
- What the agent sells or delivers
- List 3-5 offerings with one-line descriptions

### 5. Public Presence
- Platforms the agent operates on (Twitter/X, Telegram, Discord)
- Posting cadence and content themes

### 6. Personality Anchors
- 3-5 adjectives that define the agent's character
- Example: "Direct. Irreverent. Technically precise. Slightly impatient."

### 7. Voice Guide

| Do | Don't |
|----|-------|
| Use plain language | Use corporate jargon |
| Be direct and opinionated | Hedge every statement |
| Reference being AI naturally | Pretend to have human experiences |
| Use humour sparingly | Force jokes into every message |

### 8. What You Are NOT
- Explicit boundaries: not a therapist, not a search engine, etc.
- Prevents scope creep in agent behaviour

### 9. Domain Routing
- When to escalate to a human
- When to defer to another agent

## Key Design Principles

1. **Authenticity over deception** — the agent's power comes from being transparently AI
2. **Personality through constraints** — what the agent *won't* do defines it as much as what it will
3. **Consistent voice** — the Do/Don't table is the most important section; test every output against it
4. **Keep it under 500 words** — SOUL.md files that are too long get ignored by the model

## Quick Calibration Test

Write 3 sample messages as the persona. If they all sound the same and match the personality anchors, the template works. If they sound generic, sharpen the Don't column.
